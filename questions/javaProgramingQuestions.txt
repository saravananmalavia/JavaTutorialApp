### **Simple Conditional Logic Questions**
1. Write a program to check if a number is **even** or **odd**.
2. Write a program to check if a number is **positive**, **negative**, or **zero**.
3. Write a program to find the **largest** of two numbers.
4. Write a program to find the **largest** of three numbers.
5. Write a program to check if a year is a **leap year**.
6. Write a program to calculate the **grade** of a student based on marks (e.g., A for >90, B for 80-89, etc.).

---

### **Basic Looping and Iterative Problems**
7. Write a program to print numbers from **1 to 100** using a loop.
8. Write a program to print the **sum of numbers from 1 to N**.
9. Write a program to print the **multiplication table** of a given number.
10. Write a program to find the **factorial** of a number.
11. Write a program to calculate the **sum of all even numbers** from 1 to N.
12. Write a program to calculate the **sum of all odd numbers** from 1 to N.
13. Write a program to count the number of **digits** in a given number.
14. Write a program to reverse a given number (e.g., 123 → 321).
15. Write a program to check if a number is a **palindrome** (e.g., 121 is a palindrome).
16. Write a program to check if a number is **prime**.
17. Write a program to print all **prime numbers** between 1 and N.
18. Write a program to calculate the sum of the **digits** of a number.

### **Intermediate Logical Problems**
22. Write a program to find the **greatest common divisor (GCD)** of two numbers.
23. Write a program to check if a number is an **Armstrong number** (e.g., 153 → \(1^3 + 5^3 + 3^3 = 153\)).
24. Write a program to generate the **Fibonacci series** up to N terms.
25. Write a program to print the sum of the following series:
    \[
    1 + \frac{1}{2} + \frac{1}{3} + \dots + \frac{1}{N}
    \]
